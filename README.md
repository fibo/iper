## Installation

With [npm](https://npmjs.org/) do

```
npm install iper
```

With [bower](http://bower.io/) do

```bash
$ bower install iper
```

## API

### new Graph([graph])

> Hypergraph

```
var graph = new Graph()
```

|                 |              |
| --------------- | ------------ |
| @param {Object} | [graph]      |
| @param {Object} | [graph.edge] |
| @param {Object} | [graph.node] |

### graph.addEdge(nodeIds)

|                   |         |
| ----------------- | ------- |
| @param {Array}    | nodeIds |
| @returns {String} | id      |

### graph.addNode(data)

|                   |      |
| ----------------- | ---- |
| @param {Any}      | data |
| @returns {String} | id   |

### graph.delEdge(id)

|                  |       |
| ---------------- | ----- |
| @param {String}  |id     |

|                 |      |
| --------------- | ---- |
