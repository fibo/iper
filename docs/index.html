iper

Hypergraphs for Node.js

Description

What is an hypergraph?

An hypergraph is a graph which rank can be greater than two.

Check the following Wikipedia pages:

  • Hypergraph
  • Graph theory

Installation

With npm do

npm install iper

Features

  • Graphs of rank n, a.k.a. hypergraphs
  • Directed graphs
  • Graph theory algorithms:
    • adjacent nodes
