Create nodes

var iper = require('iper')
  , should = require('should');

var IperGraph = iper.IperGraph;

Create an empty graph, and create a node

var graph = new IperGraph();
graph.createNode();

Every node will be given a unique identifier, that is returned by #createNode() as a convenience. Lets create two nodes and store ids to use them later.

var id1 = graph.createNode([]);
var id2 = graph.createNode([]);

Now we can use id1 and id2 to refer to the nodes, for example we can create an edge joining the nodes.

graph.createEdge([id1, id2]);

We can also get a reference to the node

var node1 = graph.getNode(id1);
